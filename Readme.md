# Lumina AI Upscaler

A premium image upscaling tool powered by Real-ESRGAN.

## Cara Menjalankan

1. Buka terminal di folder ini.
2. Jalankan server dengan perintah:
   ```bash
   .\venv\Scripts\python.exe main.py
   ```
3. Buka browser dan akses:
   ```
   http://localhost:8000
   ```

## Fitur
- Upscale hingga 4x Ultra HD.
- Perbandingan Side-by-Side (Slider).
- Zoom Preview untuk melihat detail pixel.
- Pilihan berbagai model AI.
